# Sorimadang Vancouver (소리마당 밴쿠버)

> Official website for Sori Madang Vancouver, offering Korean traditional music and Pansori classes.
> 캐나다 밴쿠버 판소리 & 한국 전통 음악 클래스 '소리마당 밴쿠버'의 공식 웹사이트입니다.

🌐 **[Live Demo](https://sorimadang-vancouver.github.io/)**

## 🌟 Features

- **Bilingual Support (한/영 지원)**: Seamlessly switch between English and Korean instantly without page reloads.
- **Dark/Light Mode (다크/라이트 모드)**: Elegant and smooth theme transitions matching user preferences.
- **Responsive Design (반응형 웹)**: Fully optimized for mobile, tablet, and desktop viewing.
- **Custom Animations (스크롤 애니메이션)**: Custom Intersection Observer-based staggered scroll reveal animations for a premium aesthetic feel.
- **Zero Dependencies (가벼운 구성)**: Built entirely with Vanilla HTML, CSS, and JavaScript for maximum performance and lightning-fast loading times. (Uses [Lucide Icons](https://lucide.dev/) for lightweight SVG iconography).

## 🚀 Setup & Local Development

Since this project uses vanilla web technologies without complex frameworks, there is no build process required. 

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/sorimadang-vancouver.git
   ```
2. Navigate to the project directory and simply open `index.html` in your web browser.
3. *(Optional)* For the best development experience, use a local development server like VS Code's **Live Server** extension.

## 🛠️ Technologies Used

- **HTML5**: Semantic and accessible document structure.
- **CSS3**: Custom design system utilizing CSS Variables for dynamic theming, CSS Grid/Flexbox for flexible layouts, and smooth transition animations.
- **Vanilla JavaScript**: DOM manipulation, `IntersectionObserver` for performant scroll animations, and `localStorage` for persisting user theme and language preferences.
